## 启动流程
- 拉取仓库
  
- 进入 **frontend** 
  
- 在终端输入<br>
  
    `npm install` 安装依赖    
    `npm run dev` 运行前端应用

- 另起一个终端  
  
- 进入项目目录    
  
    `go mod tidy` 安装依赖包    
    `go run main.go` 启动后端应用

- 记得修改 **config.yaml** 文件
  
- 浏览器中输入地址 **http://localhost:3000**
